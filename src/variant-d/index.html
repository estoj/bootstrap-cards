UX report

Variant D

Introduction

This analysis focuses exclusively on the card from Variant A, based on the video and screenshot that was provided.
Given the lack of content for this element, this report is necessarily limited by an absence of context regarding its use, particularly in terms of information hierarchy and the relevance or not of such a layout.

Strengths

  1. Consistency: The layout, spacing and color consistency across tabs allow good predictability for users, which gives a smooth navigation.
  2. Responsive design: The component adjusts appropriately for different viewport sizes, delivering a slick experience on most used screen sizes.

Usability Issues

Despite adhering to basic usability principles, this component still has several weaknesses.

  1. Visual Hierarchy (severity: high)
    This component lacks clear visual prioritization, notably between title and content font sizes, with the main visual differences between all elements being almost exclusively color (the 'primary' color having relatively limited contrast with the regular text color) and font weight.
  2. Interactive Feedback (severity: high)
    All buttons lack clear hover states to indicate interactivity and using the primary color on a white or light gray background does not give enough predictability.
  3. Information Hierarchy (severity: medium)
    Despite a basic structure using tabs, this component does not allow to fully identify its key elements, namely its purpose, its interactive elements (as mentioned in the previous point) as well as a good vision of its usefulness.
    Also, using a list of buttons to navigate between the 3 images is also not ergonomically appropriate.

Redesigning the element

While this card implementation provides a functional layout, several usability issues hinder the overall user experience.
Therefore, I tried to fix the negative points, including:

  • Improving visual contrast with a better font size scale.
  • Improving the affordance of interactive elements, with a clearer hover state.
  • Revising the functionality of the image selection buttons by transforming this section into a slider, which eliminates the need to move the cursor to change images, and avoids having an endless list as the image list grows longer.

You can see the result of the overall on the redesigned card page.
